# rtt-bsp-hpm6200evk

RT-Thread Studio BSP for HPM6200EVK Board

## Supported examples

***

- blink_led
- uart_dma_demo
- timer_demo
- flashdb_demo
- mcan_example
- adc_example
  - Valid voltage range is [0V, 3.3V], make sure the input voltage is in valid range, otherwise the SoC may be demaged.
- usb_device_generic_hid
- usb_host_msc_udisk
